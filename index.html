An interactive Story

Leveraging data from the Bureau of Transportation Statistics We can investigate per airline their average on-time arrivals and causes for delay by airline. This martini-glass structured narrative visualization follows Schneiderman's mantra and provides an overview of the dataset first. This dataset provides per carrier per location from 2018-05 through 2019-05 the number of delays and the number of minutes for each type of delay.
